# INVITEA

Plataforma web de invitaciones digitales para eventos sociales.

## Tecnologías

* Next.js
* React
* TypeScript
* Tailwind CSS
* Supabase
* Vercel

## Objetivo

Permitir a los usuarios crear, personalizar y administrar invitaciones digitales modernas con RSVP, dashboard y estadísticas.

## Estado actual

Proyecto inicializado correctamente.
